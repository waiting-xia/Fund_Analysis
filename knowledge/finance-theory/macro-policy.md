# 从变量到资产的传导链

宏观分析的重点不是给单个数据贴“利好”或“利空”标签，而是识别政策、增长、通胀、流动性和风险偏好之间的传导链。理论关系会受到市场预期和估值位置影响，同一数据在不同阶段可能产生不同价格反应。

# 货币政策与利率

货币政策通过政策利率、公开市场操作、准备金、信贷条件和预期管理影响资金成本。宽松政策通常有利于流动性和估值，但若宽松来自增长显著恶化，盈利下修可能抵消估值扩张。债券价格与收益率方向相反，久期越长，对利率变化通常越敏感。

收益率曲线包含市场对未来短端利率、增长、通胀和期限溢价的综合预期。曲线陡峭化和扁平化可能由不同端点驱动，必须区分“增长改善导致长端上升”和“宽松预期导致短端下降”等路径。

# 财政政策与信用周期

财政政策通过政府支出、税收、转移支付和专项融资影响总需求。政策效果取决于资金落地速度、乘数效应、地方财政约束和私人部门信心。信用扩张通常先影响融资与投资，再传导至产出和价格；社融或信贷增长也要结合结构、基数和实体融资需求解读。

信用周期上行可能改善周期行业盈利和风险偏好，但杠杆积累会增加后续脆弱性。信用收缩阶段，低质量债务、弱现金流企业和低流动性资产通常承压更大。

# 通胀、实际利率与风格

通胀由需求、供给、工资、商品价格和预期共同决定。名义利率减通胀预期可近似实际利率。实际利率上升通常提高远期现金流折现率，对高估值长久期资产压力更大；但若上升来自强劲真实增长，盈利改善可能部分对冲。

温和通胀环境下，具备定价权的企业更容易传导成本。高通胀与增长放缓并存时，传统股债分散效果可能下降，需要重视商品、现金流质量和久期风险。

# 汇率与跨境资产

汇率受利差、贸易、资本流动、风险偏好和政策预期共同影响。人民币升值或贬值对行业影响并不一致：进口成本、出口收入、外币负债和海外资产折算方向各异。QDII 基金还需区分底层资产收益和汇率贡献，是否对冲汇率会改变结果。

# 宏观情景分析

可用“增长上行/下行”和“通胀上行/下行”构造基础情景矩阵，再叠加政策反应、估值和市场预期。每个结论都应说明触发指标和失效条件，例如盈利预期、信用脉冲、实际利率、库存、汇率和风险溢价。宏观变量适合解释风险来源，不能替代基金持仓与净值的最新事实。
