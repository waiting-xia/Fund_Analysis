# 适用边界

本条目是通用研究框架，不包含任何基金的当前行情，也不构成投资建议。基金分析应把正式披露事实、确定性计算结果和研究推断分开。净值、份额、规模、持仓和基金经理信息必须保留来源及日期。

# 净值与收益口径

单位净值反映每份基金资产的账面价值，累计净值通常把历史分红因素纳入，更适合观察长期财富变化。比较不同份额类别时，应注意申购费、销售服务费、管理费和托管费的差异。区间收益必须说明起止日、分红处理、币种和是否年化；短期估算净值不能代替基金管理人披露的正式净值。

货币基金常使用每万份收益和七日年化收益率，不能直接按普通单位净值基金的方式解读。ETF 还要区分基金净值、交易价格和折溢价。QDII 基金受海外交易时区、汇率和净值披露时滞影响，当日场内价格与最近正式净值可能不在同一信息时点。

# 主动与被动基金评价

被动指数基金的核心是复制质量。常用指标包括跟踪误差、跟踪偏离度、费率、流动性、规模及折溢价。跟踪误差衡量相对基准收益差的波动，长期偏离还可能来自费用、现金拖累、申赎冲击和成分股调整。

主动基金应把绝对收益与相对基准、同类排名、承担的风险和市场风格放在一起。超额收益不等于稳定能力；需要观察不同市场阶段的一致性、回撤修复、行业集中度、个股选择与仓位变化。基金经理任期短于评价窗口时，不应把全部历史业绩归因给现任经理。

# 持仓、换手与风格漂移

公开持仓通常来自季报、半年报或年报，存在披露时滞。季报往往只披露前十大重仓，不能据此还原完整组合，更不能把两个报告期之间的权重变化直接解释为某一天的买卖。权重变化同时受主动交易、股票涨跌、申赎、分红和组合总规模变化影响。

换手率较高可能代表主动捕捉机会，也可能带来交易成本和风格不稳定。判断风格漂移时，可联合观察行业权重、持仓估值、大小盘暴露、因子收益敏感度和基金名称所承诺的投资范围。单个报告期的变化只能视为线索，应与多个周期交叉验证。

# 基金经理与治理

基金经理评价可从任期、共同管理安排、投资框架稳定性、能力圈、回撤应对和公司投研平台等方面展开。规模快速增长会改变策略容量，尤其影响小盘、量化或低流动性策略。基金公司治理、人员变动、产品复制和激励约束也会影响策略持续性。

# 研究结论结构

专业结论至少回答四个问题：收益来自何种风险暴露或选择能力；当前组合可能受哪些宏观和行业变量影响；风险在什么场景下集中暴露；哪些可观察条件会验证或推翻原判断。任何操作建议都应附带适用期限、风险预算和退出条件，而不是只给出方向性标签。
