# 因子是风险与行为的统计表达

因子用于描述一组资产共同的收益特征。因子收益可能来自风险补偿、行为偏差或制度约束，不是永久有效的公式。任何因子评分都必须说明指标定义、数据时点、标准化方式、权重、再平衡频率和交易成本。

# 价值因子

价值因子偏好相对基本面价格较低的资产，常用 PE、PB、现金流收益率和股息率等指标。亏损公司、周期盈利高点和行业会计差异会扭曲单一指标，因此需要多指标合成、行业中性化和质量过滤。价值陷阱通常伴随盈利持续下修、资产负债表恶化或商业模式衰退。

# 动量因子

动量因子认为近期相对强势资产在一段时间内可能延续，常用过去若干月收益并跳过最近极短窗口。动量在趋势反转时可能发生快速回撤，且容易拥挤。用于基金分析时，应区分基金净值动量、底层持仓动量和申购资金追涨，它们不是同一概念。

# 质量与低波动因子

质量因子常考察盈利能力、现金流质量、杠杆、利润稳定性和治理。高质量不代表任何价格都值得买入，仍需结合估值。低波动因子偏好历史波动或贝塔较低资产，但可能集中于利率敏感行业，并在市场快速反弹时落后。

# 规模与流动性

小市值因子可能获得成长和风险补偿，也承担更高的流动性、治理和交易成本风险。基金规模与底层资产容量需要匹配：策略规模扩大后，换手、冲击成本和持仓集中度可能改变原有收益特征。

# 多因子合成

多因子模型通常先对原始指标去极值、标准化，再按行业或市值中性化，最后组合。等权简单透明，但忽略因子稳定性和相关性；动态权重可能适应市场，却增加过拟合。模型评估应使用样本外数据，纳入交易成本，并检查因子之间是否重复计量同一风险。

基金五因子评分可以覆盖市场状态、估值、动量、风险、质量与资金，但总分不能隐藏子项冲突。例如高动量与高估值同时出现时，结论应明确收益延续与估值回撤两种情景。评分只用于排序和观察，不应直接变成自动交易指令。

# 拥挤、失效与解释

当大量资金使用相似信号时，因子可能变得拥挤，退出阶段的相关性和冲击成本会上升。因子短期失效不必然证明理论永久消失，也可能代表市场状态变化。解释基金信号时，应给出数据置信度、主要贡献因子、反向指标和下一次复核条件。
