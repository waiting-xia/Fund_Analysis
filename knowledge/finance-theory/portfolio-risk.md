# 风险不是单一数字

本条目提供一般性风险分析方法。波动率、回撤、相关性和风险调整收益分别描述风险的不同侧面，不能互相替代。历史统计依赖样本窗口和数据频率，不代表未来分布保持不变。

# 波动率与下行风险

年化波动率通常由周期收益率的标准差乘以年化系数得到。日频数据常使用平方根时间法则，但它隐含收益独立同分布等近似假设，在波动聚集、跳跃和长假期间可能失真。比较基金时应保持相同的数据频率、窗口和净值口径。

投资者通常更关心损失而非上涨波动。下行波动率只统计低于目标收益或零收益的部分，索提诺比率以超额收益除以下行风险。它能补充夏普比率，但依然对窗口和目标收益敏感。

# 最大回撤与修复

最大回撤是净值从历史高点到随后低点的最大跌幅。它直观体现持有体验，却只描述样本中最差的一段，不能表示所有尾部风险。分析时应同时记录回撤开始、谷底、持续时间和恢复到前高所需时间。

卡玛比率常用年化收益除以最大回撤。较高数值可能来自稳定收益，也可能只是样本未覆盖压力阶段。对于成立时间短或净值过度平滑的资产，卡玛比率可能显得虚高。

# 相关性、协方差与分散化

两资产的组合风险取决于各自波动率、权重和协方差。低相关资产在常态下可以分散风险，但相关性会随市场状态变化：流动性紧张和系统性冲击发生时，不同风险资产的相关性往往上升。因此应观察滚动相关性，并对压力情景设置更保守的相关假设。

基金名称不同不代表风险来源不同。两只基金可能持有不同股票，但同时暴露于成长、利率敏感或小盘流动性因子。组合分散应穿透到行业、地区、久期、信用、货币和风格因子，而不能只按产品数量判断。

# 风险调整收益

夏普比率以超过无风险收益的组合收益除以总波动率，适合比较相近口径的策略。它会把上涨和下跌波动同等处理，也难以完整描述偏度、厚尾和路径依赖。信息比率则以相对基准的主动收益除以跟踪误差，更适合评价主动管理的稳定超额。

计算风险调整收益时，无风险利率、基准和年化方式必须一致。高收益但低流动性的资产可能因报价不连续而表现出虚假的低波动，应额外考虑成交能力、赎回约束和估值平滑。

# 压力测试与风险预算

压力测试不是精确预测，而是评估组合对极端但合理情景的脆弱性。可设置利率快速上升、信用利差扩大、权益估值压缩、汇率大幅波动、行业政策变化和赎回冲击等情景。风险预算应把允许损失、资金期限和再平衡规则转化为可执行约束，并在相关性上升时留出安全余量。
